# Jacode 사용자 가이드

이 문서는 Jacode 플랫폼의 일반 사용자를 위한 가이드입니다. 프로젝트 생성부터 AI 코드 생성 기능 활용까지 개발 워크플로우 전반을 다룹니다.

## 1. 시작하기

### 로그인 및 대시보드

Jacode에 로그인하면 메인 대시보드로 이동합니다. 대시보드에서는 다음을 확인할 수 있습니다.

- **최근 프로젝트**: 최근에 작업한 프로젝트 목록
- **시스템 상태**: AI 서비스 연결 상태 및 주요 지표
- **활동 로그**: 최근 수행된 작업 내역

### 프로젝트 생성

새로운 프로젝트를 시작하려면:

1. 대시보드 우측 상단의 **"+ 새 프로젝트"** 버튼을 클릭합니다.
2. 프로젝트 이름과 설명을 입력합니다.
3. 프로젝트가 생성되면 자동으로 에디터 화면으로 이동합니다.

---

## 2. 에디터 사용법 (Editor)

Jacode의 에디터는 VS Code와 유사한 사용자 경험을 제공하는 Monaco Editor를 기반으로 합니다.

### 파일 탐색기

- 좌측 사이드바에서 프로젝트의 파일 트리를 볼 수 있습니다.
- 파일/폴더 생성, 이름 변경, 삭제가 가능합니다.
- 파일을 클릭하면 메인 에디터 영역에 열립니다.

### 코드 편집

- **구문 강조 (Syntax Highlighting)**: 주요 프로그래밍 언어를 지원합니다.
- **자동 완성**: 기본적인 코드 자동 완성을 지원하며, AI가 추가적인 제안을 제공할 수 있습니다.
- **미니맵**: 긴 코드의 개요를 우측에서 확인할 수 있습니다.

---

## 3. AI 기능 활용 (AI Features)

Jacode의 핵심 기능인 AI 코딩 어시스턴트를 활용하는 방법입니다.

### AI 채팅 (Chat)

우측 패널의 'Chat' 탭에서 AI와 대화하며 코딩 도움을 받을 수 있습니다.

- **질문하기**: "이 함수가 무슨 역할을 해?", "React로 로그인 폼 만들어줘" 등 자연어로 질문하세요.
- **컨텍스트 인식**: 현재 열려있는 파일을 자동으로 인식하여 더 정확한 답변을 제공합니다.
- **코드 삽입**: 채팅에서 생성된 코드는 'Insert' 버튼으로 에디터 커서 위치에 바로 삽입할 수 있습니다.

### AI 코드 생성 (Generate)

복잡한 기능을 구현할 때 AI 생성 툴을 사용하세요.

1. 우측 패널의 **Create/Edit** 탭을 선택합니다.
2. **프롬프트 입력**: 구현하고자 하는 기능을 설명합니다. (예: "JWT 인증 미들웨어 작성해줘")
3. **설계 확인 (Blueprint)**: AI가 먼저 구현 계획(Blueprint)을 제시합니다. 내용을 확인하고 수정할 수 있습니다.
4. **코드 생성**: 계획을 승인하면 AI가 실제 코드를 생성합니다.
5. **Diff 보기 & 적용**: 생성된 코드를 기존 코드와 비교(Diff View)하여 변경 사항을 확인한 후 적용(Apply)하거나 거절(Reject)할 수 있습니다.

### 인라인 힌트 & 품질 검사

- 에디터 내에서 문제가 있는 코드나 개선이 필요한 부분에 물결 밑줄이나 전구 아이콘이 표시될 수 있습니다.
- AI가 제안하는 리팩토링이나 버그 수정을 확인하고 적용하세요.

---

## 4. 단축키 (Shortcuts)

- `Ctrl + S`: 저장
- `Ctrl + P`: 파일 빠른 열기
- `Ctrl + Shift + F`: 전체 검색
- `Ctrl + '`: 터미널 열기/닫기 (지원 예정)

## 5. 문제 해결

- **AI 응답이 느린 경우**: 복잡한 작업이나 네트워크 상태에 따라 시간이 소요될 수 있습니다. 상단 상태 표시줄의 로딩 인디케이터를 확인하세요.
- **에러 발생**: 우측 하단의 알림 메시지를 확인하고, 문제가 지속되면 관리자에게 문의하세요.
